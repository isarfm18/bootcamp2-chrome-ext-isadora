Resumindo — Link Saver

Salve a aba atual e links do botão direito em uma lista rápida dentro do popup.

Instalação

  1. Acesse chrome://extensions e ative Developer mode.
  2. Clique em Load unpacked e selecione a pasta do projeto.
  3. Abra o popup da extensão.

Uso

  • Salvar aba atual via popup.
  • Salvar link ou página via botão direito (context menu).
  • Filtrar, remover, copiar ou exportar links no popup.
